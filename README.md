# Grupo Cordillera Frontend

Frontend desarrollado con React y Vite para la gestión e integración con el Backend For Frontend (BFF) y microservicios del sistema Grupo Cordillera.

## Tecnologías utilizadas

* React
* Vite
* Axios
* Node.js
* NPM

## Requisitos

* Node.js instalado
* NPM instalado

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

El proyecto se ejecutará en:

```text
http://localhost:5173
```

## Estructura del proyecto

```text
src/
├── components/
├── pages/
├── services/
```

## Integración

El frontend consume servicios del BFF mediante Axios.

Puerto utilizado por el BFF:

```text
http://localhost:8080
```
